Les méthodes de tableaux

Définition : Les méthodes de tableaux sont des fonctions qui permettent de manipuler les tableaux. il existe plusieurs méthodes de tableaux, nous allons en voir quelques unes: notamment :

  • push()
  • pop()
  • shift()
  • unshift()
  • splice()
  • slice()
  • sort()
  • reverse()
  • join()
  • concat()
  • indexOf()
  • lastIndexOf()
  • forEach()
  • map()
  • filter()
  • reduce()
  • reduceRight()
  • every()
  • some()
